# chat-with-pdf
Gradio app on python and Anthropic, OpenAI, LLamaIndex.
Tested on Python 3.12


1. Add .env file with required API Keys
```
ANTHROPIC_API_KEY=
OPENAI_API_KEY=
LLAMA_CLOUD_API_KEY=
```
2. Install Requirements
```
pip install -r requirements.txt
```
3. Run Application
```
python app.py
```
